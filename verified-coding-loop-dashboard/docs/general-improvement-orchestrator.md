# General Improvement Orchestrator

This layer combines the major learning signals into one balanced next action.

## Inputs

```text
verified learning state
meta learning state
compute credit state
process focus report
efficiency optimizer report
latest improvement proposal
```

## Score dimensions

```text
proven quality
learning momentum
focused target strength
efficient next step
reserve strength
proposal readiness
```

## Output

The orchestrator writes a general improvement report with:

```text
general score
selected move
ranked moves
state update
next human step
```

## Purpose

The system should not only optimize one narrow layer.

It should combine the full picture and choose a balanced next improvement that is useful, small, reviewable, and efficient.
