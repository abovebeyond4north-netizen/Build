# Process Focus System

This layer breaks the learning loop into individual processes, subprocesses, and focus factors.

## Focus factors

```text
clarity
feedback
verification
reversibility
cost control
skill growth
usefulness
```

## Process groups

```text
quality verification
architecture integrity
utility correctness
schema integrity
proposal generation
experience and value learning
meta learning
credit control
```

Each process contains smaller subprocesses. Each subprocess has factor scores from 0 to 1.

## Experience function

The experience function computes a weighted score for each subprocess.

The system then ranks the weakest subprocesses and creates recommendations.

## Why this matters

The loop no longer improves vaguely.

It now asks:

```text
Which process is weak?
Which subprocess is weak?
Which factor is weak?
What small reversible change should improve it?
```

## Safe boundary

This system produces internal reports and reviewable recommendations only.
