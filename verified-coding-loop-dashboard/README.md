# Verified Coding Loop Dashboard

A React + Tailwind dashboard that demonstrates a safe, test-driven coding improvement loop using simulated data only.

> This is a functional software engineering dashboard. It does not claim consciousness, sentience, or autonomous self-modification.

## What it demonstrates

- System state tracking
- Append-only event logging
- Episodic, semantic, procedural, and meta memory
- Belief tracking with confidence and evidence
- Prediction before action
- Prediction error after observation
- Benchmark results
- Simulated patch candidates
- Verification pipeline
- Oracle gate approval and rejection
- Snapshot and rollback flow
- Curriculum growth
- Syntax genome for Python learning areas

## Safety boundaries

This first version intentionally avoids risky behavior:

- No real code execution
- No network automation
- No autonomous deployment
- No backend
- No external side effects
- Local React component state only

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
